from IPFS

from GitHub

If you need the source, please don't directly download the source; clone it. (you can download as compressed file, but you will need manual work to update Floflis, like on the Lite version).

Clone:

SSH

git clone git@gitlab.com:floflis/Floflis-DNA.git or git clone git@github.com:Floflis/Floflis-DNA.git

HTTP

git clone https://gitlab.com/floflis/Floflis-DNA.git or git clone https://github.com/Floflis/Floflis-DNA.git
